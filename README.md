Welcome to my **Tailwind CSS** learning repository! This project is a personal space dedicated to mastering [Tailwind CSS](https://tailwindcss.com/), a utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.

## 🎯 Objective
- Understand the utility-first CSS methodology.
- Master responsive design, state variants (hover, focus), and Dark Mode.
- Learn how to customize and extend the default design system via `tailwind.config.js`.
- Practice building clean, modern, and responsive UI components from scratch.

## 🚀 Installation & Versions

This repository focuses on **Tailwind CSS v3.x** (with an eye on the upcoming features in v4.x).
